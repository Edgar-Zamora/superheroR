The `superheroR` package is a R wrapper that is used to request data
from the [SuperHero API](https://superheroapi.com) by specific
superhero. The API provides data about a superheros stats, biography,
appearances, and more from over 500 superheros from both the comic
universes. The only requirement is a access token which can be obtained
by linking an individuals personal Facebook account.

# Setting access token

``` r
Sys.setenv(SUPERHERO_TOKEN = "xxxxxxxxxxxxxxx")
```

# Installation

``` r
## install.packages(devtools)

devtools::install_github("Edgar-Zamora/superheroR")
```
